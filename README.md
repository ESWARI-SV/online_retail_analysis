# Online Retail Analytics (Python + Power BI)

## Overview
This project analyzes an Online Retail dataset (Kaggle) to uncover sales trends, return behavior, top products, and customer segments using RFM analysis.

## Tools Used
- Python (Pandas, NumPy)
- Power BI (DAX, Data Modeling, Dashboarding)

## Dataset Columns
- InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country

## Key Steps
- Data cleaning (missing values, duplicates, returns)
- Feature engineering (Revenue, Returns, Year-Month)
- Created Orders Table and RFM Table
- RFM Segmentation (Champions, Loyal, At Risk, Lost, New, Others)
- Built interactive Power BI dashboard

## Dashboard KPIs
- Gross Revenue
- Total Orders
- Total Customers
- Average Order Value (AOV)
- Return Rate %
